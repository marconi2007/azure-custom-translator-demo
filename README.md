# Azure Custom Translator - Projeto Demo

Projeto demonstrativo simulando integração com Azure AI Translator (Custom Translator).

## Objetivo

Demonstrar arquitetura de consumo da API de tradução personalizada da Azure.

## Arquitetura

Cliente → API FastAPI → Serviço de Tradução → Azure Translator (Simulado)

## Tecnologias

- Python
- FastAPI
- Azure AI Translator (conceito)

## Observação

Este projeto utiliza uma simulação da API do Azure para fins acadêmicos/demonstrativos.
